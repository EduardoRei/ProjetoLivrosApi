# ProjetoLivrosApi
Este é um projeto feito apenas com Html,Css,JavaScript. Que consome livros especificadas através da url de uma api, e apresenta em uma tela, demonstrando o titulo e o nome do autor. A tela é interativa, ela permite que você exclua um livro da lista e insira outros livros. O programa esta usando o LocalStorage(memória do cache do seu navegador) para armazenar os dados.

![Imagem da tela](/img/Tela.png)

# Como executar
Para executar é muito simples:
- Clone o repósitorio.
- Identifique o arquivo index.html. 
- Abra o arquivo
- E com apenas estes três passos ja da para interagir com o programa
